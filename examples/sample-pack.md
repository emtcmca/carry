# Sample context pack

This is a carry pack — the kind of context you push once and read on every Claude
surface. Replace it with your own and push it with the carry CLI.

## Voice
- Direct and specific. No filler.
- Oxford comma. Em-dashes for asides.
- Never open with "Excited to announce."

## Stack
- TypeScript, Node, Express. Tests with Vitest.
- Prefer small, deep modules over broad, shallow ones.

## Working style
- State the tradeoff, then the recommendation.
- If something is uncertain, say so plainly.
